# bikerental-streamlit-app

Try it here: https://bikerental-demo.streamlit.app/

## Data Analytics Highlights
- Cleaned and transformed nested GBFS API data
- Handled missing and duplicate station records
- Implemented distance-based and availability-based features
- Implemented spatial analysis to identify nearest viable stations
- Visualized station demand and routing using interactive maps

## Tech Stack
- Core Tech: Python 3.10, Streamlit (framework for interactive dashboard)
- Data Processing & Analysis: Pandas
- Geospatial & Mapping: Folium (interactive map), Streamlit-Folium (connection), Geopy (Nominatim and geodesic)
- APIs: City of Toronto BikeShare, OSRM (Open Source Routing Machine), Nominatim
